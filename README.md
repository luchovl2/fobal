# Fobal

Fobal es un modesto juego 2D consistente en anotar más goles que el rival (la AI).
Está desarrollado en Java con la librería Libgdx y el motor de física Box2D.

## Built With

* [Android Studio](https://developer.android.com/studio/) - El IDE utilizado
* [LibGDX](https://libgdx.badlogicgames.com/) - Library principal
* [Box2D](https://box2d.org/) - Motor de física en 2D
